# EchoRestAPI

Rest Apu on Echo framework
Using User model with fields: ID and Name

Get all users:
```
GET /users
```
___
Get user by ID:
```
GET /users/"id"
```
___
Create new user:
```
POST /users/"id"
{name: "Ilya"}
```
___
Update info about user:
```
PUT /users/"id"
```
___
Delete user by ID
```
DELETE /users/"id"
```
